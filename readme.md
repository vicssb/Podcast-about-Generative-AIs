# Course: Creating Podcast Generated by A.I.s

This repository contains materials developed during the course "Project Podcast Generated by A.I.s"

## Description

Project aimed at generating a podcast using AI tools through more refined prompts.

Uses a prompt pipeline to produce each stage of the creative process.

## Technologies used in the project
- [ChatGPT](https://chat.openai.com/)
- [MidJourney](https://www.midjourney.com/app/)
- [ElevenLabs](https://beta.elevenlabs.io/)
- [Capcut](https://www.capcut.com/pt-br/)

## ✨ How it was done?

- Script generated via ChatGPT
- Audio generated by ElevenLabs
- MidJourney to create cover images
- Capcut to process audio and add background sounds
